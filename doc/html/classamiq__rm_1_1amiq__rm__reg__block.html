Detailed Description

This class is used to model a register block. The register block contains the instances of all registers. It can also contain the instances of all address maps (or just the top level address maps). The register block is the solution of having the registers visible in all address maps. Each address map contains a pointer to a reg_block. If an hierarchical implementation is used then the pointer should be connected and propagated down the hierarchy (to have visibility to the registers).
